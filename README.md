# 🐉 Legal pages

## _Проект на HTML + css, реализующий обязательные юридические страницы веб-приложения для системы инвентаризации логистического/торгового микро-предприятия._

## 📚 Описание проекта

Данный проект представляет собой отдельный модуль клиентской части системы, в котором реализованы следующие страницы:

- [Terms of Usage]
- [Privacy Policy]
- [Public Offer]

---

## 🛠 Технологии

- HTML + css
- Markdown

---

## 🚀 Быстрый старт

1. Клонировать репозиторий:

   ```bash
   git clone https://github.com/your-username/site-legal.git

   ```

2. Установить зависимости:

   ```bash
   npm install

   ```

3. Запустить проект в режиме разработки:
   ```bash
   npm run dev
   ```
